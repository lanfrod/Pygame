Проект: “Happy Bird”

Авторы проекта: Цветков Иосиф, Узьков Павел, Татаркин Андрей

Идея: Создать игру подобную популярному 8-битному хиту Flappy Bird с дополнительными механиками и идеями.
Реализация: Весь код программы реализован в файле “game.py”. Файл состоит из 3 функций - трёх окон.
(окно игры, окно главное, окно настроек). Изображения сохраняются в файл “img”, звуки в “sounds”, а рекорд сохраняется в best.txt

Полезные добавления​:
•Реализация таблицы лидеров​
•Добавление монеток, чтобы был магазин скинов

Возможные улучшения:
•Анимированный бг, для трёхмерности​
•Сделать более приятный дизайн
